  🚀 方案一：GitHub 自动编译（强烈推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⏱ 总用时：12分钟
💰 费用：完全免费
🔧 需要工具：仅需浏览器

【第 1 步】创建 GitHub 账号（2分钟）
─────────────────────────────────
1. 打开浏览器，访问：https://github.com
2. 点击右上角 "Sign up"（如果已有账号直接登录）
3. 输入邮箱、密码、用户名
4. 完成验证
5. ✅ 登录成功


【第 2 步】上传项目（3分钟）
─────────────────────────────────
1. 登录后，点击右上角 "+" 图标
2. 选择 "New repository"（新建仓库）
3. 填写信息：
   Repository name: scanner-app
   Description: 扫描APP
   选择：Public 或 Private 都可以
4. 点击 "Create repository"（创建仓库）

5. 在新页面，点击 "uploading an existing file"
   （或 "Add file" → "Upload files"）

6. 📦 上传文件：
   → 打开文件夹：d:\skills\saolei
   → 选中所有文件和文件夹（Ctrl+A）
   → 拖拽到 GitHub 网页的上传区域
   
   ⚠️ 注意：必须上传所有文件，包括：
   ✓ app 文件夹
   ✓ .github 文件夹
   ✓ gradle 文件夹
   ✓ 所有 .gradle、.bat 等文件

7. 等待上传完成
8. 在页面底部的 "Commit changes" 按钮上方输入：
   Initial commit
9. 点击绿色按钮 "Commit changes"
10. ✅ 上传完成！


【第 3 步】自动编译（8分钟）
─────────────────────────────────
上传完成后，GitHub 会自动开始编译！

1. 点击页面顶部的 "Actions" 标签
2. 你会看到一个正在运行的工作流（黄色圆点⚪）
3. 点击进入查看详细进度
4. 等待所有步骤变成绿色 ✅（约8分钟）

编译过程：
  ⚪ → ✅ Checkout code
  ⚪ → ✅ Set up JDK 11
  ⚪ → ✅ Build Debug APK
  ⚪ → ✅ Upload APK
  ⚪ → ✅ Create Release


【第 4 步】下载 APK（1分钟）
─────────────────────────────────
编译完成后，下载 APK：

方式 A：从 Releases 下载（推荐）
  1. 回到仓库首页
  2. 右侧找到 "Releases"
  3. 点击最新的 Release（v1.0.x）
  4. 下载 "app-debug.apk"
  5. ✅ 下载完成！

方式 B：从 Artifacts 下载
  1. 在 Actions 页面
  2. 滚动到最底部
  3. 找到 "Artifacts" 区域
  4. 点击 "scanner-app-debug" 下载
  5. 解压 ZIP 文件
  6. 得到 app-debug.apk


【第 5 步】安装到手机
─────────────────────────────────
1. 将 app-debug.apk 传到手机
   （通过 USB、微信、网盘等）
2. 在手机上点击 APK 文件
3. 允许"未知来源"安装
4. 点击"安装"
5. ✅ 安装完成！

  ❓ 常见问题
════════════════════════════════════════════════════════

Q：GitHub 是什么？需要付费吗？
A：GitHub 是全球最大的代码托管平台，完全免费！

Q：上传文件卡住了怎么办？
A：可能文件较多，多等一会。或者使用 scanner-app-upload.zip 上传

Q：编译失败了怎么办？
A：查看 Actions 日志，找到红色❌的步骤，看错误信息。
   大多数情况是依赖下载问题，重新运行即可。

Q：手机无法安装 APK？
A：需要允许"未知来源"安装。
   设置 → 安全 → 未知来源 → 打开

Q：以后修改代码怎么办？
A：在 GitHub 网页上直接编辑代码，提交后自动重新编译！
