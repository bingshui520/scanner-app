# 📱 在线编译指南 - 无需安装任何工具

由于您没有 Android Studio，这里提供 **3 种完全在线** 的 APK 编译方案，无需安装任何开发工具！

---

## 🌟 方案一：GitHub Actions 在线编译（推荐）

### 优点
✅ 完全免费  
✅ 自动化编译  
✅ 无需本地环境  
✅ 编译速度快（5-10分钟）

### 步骤

#### 1. 上传项目到 GitHub

如果没有 GitHub 账号：
- 访问 https://github.com
- 注册账号（免费）

创建仓库并上传代码：

```bash
# 在项目目录执行
cd d:\skills\saolei

# 初始化 Git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 关联远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/scanner-app.git
git push -u origin main
```

#### 2. 创建 GitHub Actions 工作流

在项目中创建文件：`.github/workflows/build.yml`

内容已为您准备好（见下方文件）

#### 3. 触发编译

- 推送代码到 GitHub
- 进入仓库页面
- 点击 "Actions" 标签
- 等待编译完成
- 下载生成的 APK

## 🔥 方案二：Replit 在线编译

### 优点
✅ 无需安装  
✅ 浏览器直接操作  
✅ 支持在线编辑  
✅ 免费套餐够用

### 步骤

#### 1. 访问 Replit
https://replit.com

#### 2. 导入项目
- 点击 "Create Repl"
- 选择 "Import from GitHub"
- 或直接上传文件夹

#### 3. 配置环境
在 `.replit` 文件中添加：
```
run = "./gradlew assembleDebug"
```

#### 4. 运行编译
- 点击 "Run" 按钮
- 等待编译完成
- 下载 APK

## 💻 方案三：Gitpod 在线开发环境

### 优点
✅ 完整的在线 IDE  
✅ 预配置开发环境  
✅ 支持 VSCode 界面  
✅ 每月 50 小时免费

### 步骤

#### 1. 访问 Gitpod
https://gitpod.io

#### 2. 使用 GitHub 登录

#### 3. 打开项目
- 在你的 GitHub 仓库 URL 前加上 `gitpod.io/#`
- 例如：`gitpod.io/#https://github.com/你的用户名/scanner-app`

#### 4. 自动配置
Gitpod 会自动：
- 安装 JDK
- 安装 Android SDK
- 配置 Gradle

#### 5. 编译 APK
在终端执行：
```bash
./gradlew assembleDebug
```

#### 6. 下载 APK
右键点击生成的 APK -> Download

## 📦 方案五：本地轻量级编译（最后选择）

如果以上方案都不行，可以安装最精简的工具：

### 需要安装（总共约 200MB）

#### 1. OpenJDK 11
```
下载地址：https://adoptium.net/
选择：JDK 11 (LTS) - Windows x64
```

#### 2. 下载后配置环境变量
```
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-11.0.x
Path 添加：%JAVA_HOME%\bin
```

#### 3. 验证安装
```bash
java -version
```

#### 4. 使用项目自带的 Gradle Wrapper
```bash
cd d:\skills\saolei
gradlew.bat assembleDebug
```
